**Library Management System**

Overview
A simple web-based application to manage books in a library. It allows users to add new books and view existing ones through an easy-to-use interface.

Features
- Main Menu: Navigation to core functions (Add Book, View Book).
- Add Book: Form to enter ISBN, Book Name, Type, Author, and Cost.
- Example: ISBN 5, Name Gitai, Author Robin Sharma, Cost 799.
- View Book: Displays all stored books.

Tech Stack
- Frontend: HTML, CSS
- Backend: Java (Servlets/JSP)
- Database: JDBC with MySQL/PostgreSQL
- Server: Apache Tomcat
